# 今日头条新闻文本分类项目
基于BERT预训练模型实现的今日头条新闻标题文本分类任务，覆盖15个新闻类别。

## 📁 项目结构
grad0_train/
├── demo1/ # 第一版基础实现
├── demo1ver2/ # 第二版优化实现
└── README.md # 项目总说明文档

## ✨ 版本说明
| 版本 | 特点 | 
| :--- | :--- |
| demo1 | 基础BERT分类实现，功能完整 | 
| demo1ver2 | 优化版，包含完整的配置说明与代码注释 | 

## 📚 数据集说明
本项目使用今日头条新闻标题数据集，共覆盖15个新闻类别。
数据集来源：[百度网盘链接](https://pan.baidu.com/s/10XRGQAIKGDI5eWLjmaB9Xg?pwd=1111)

## 🚀 快速开始
以`demo1ver2`为例：
1.  进入项目目录：`cd demo1ver2`
2.  安装依赖（如果需要）
3.  运行主程序：`python main.py`
